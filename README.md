This project focuses on the development of a prototype THEOS-3 communication and switching control architecture 
based on the CubeSat Space Protocol (CSP). The system is designed to provide reliable communication 
between satellite subsystems, including the Electrical Power Subsystem (EPS) and On-Board Computer (OBC)

The implementation utilizes CAN Bus as the underlying transport layer and FreeRTOS for real-time task management. 
The project investigates the integration of CSP networking concepts into embedded systems, enabling packet routing, 
service-oriented communication, and fault-tolerant subsystem interaction.

Key Features:

- CubeSat Space Protocol (CSP) implementation
- CAN Bus transport interface
- Real-time operation using FreeRTOS
- Modular subsystem communication architecture
- Telemetry and command packet exchange
- Embedded software development for satellite applications

This project serves as a practical platform for studying spacecraft network architectures and developing embedded systems for future CubeSat missions.
